# BurningSpace

BurningSpace - локальный прототип браузерной космической аркады на Phaser 3, TypeScript и Vite.

Проект подготовлен к будущей мультиплеерной архитектуре и использует структуру монорепозитория. Текущая игровая логика пока остается локальной на клиенте: игрок, NPC, астероиды, снаряды, здоровье, базы, защита баз, HUD и админская панель работают в браузере как раньше.

## Структура

```text
apps/client      Phaser/Vite клиент и текущие игровые ассеты
apps/server      Colyseus сервер с health endpoint и комнатой battle
packages/shared  общие типы, константы и названия сетевых сообщений
```

Сервер на этом этапе не содержит движения, снарядов, астероидов, NPC, урона или переноса боевой логики. Он синхронизирует только список участников тестовой комнаты.

## Установка

```bash
npm install
```

## Запуск

Клиент:

```bash
npm run dev:client
```

Сервер:

```bash
npm run dev:server
```

Health endpoint сервера:

```text
http://localhost:2567/health
```

Ожидаемый ответ:

```json
{
  "ok": true,
  "service": "burningspace-server"
}
```

## Базовый сетевой тест

1. Запустить сервер: `npm run dev:server`.
2. Запустить клиент: `npm run dev:client`.
3. Открыть две вкладки клиента.
4. На обеих вкладках нажать `Connect`.
5. Задать разные ники.
6. Выбрать разные режимы или фракции.
7. Нажать `Apply profile`.
8. Проверить, что обе вкладки показывают одинаковый список участников.
9. Закрыть одну вкладку или нажать `Disconnect`.
10. Проверить, что участник исчез во второй вкладке.

Кнопка `Open local prototype` запускает текущую локальную Phaser-игру. Механика боя остается клиентской и пока не синхронизируется через сервер.

## Проверка

```bash
npm run typecheck
npm run build
```

Отдельно:

```bash
npm run typecheck:client
npm run typecheck:server
npm run build:client
npm run build:server
```

## Текущий статус

- Стабильный тег локального прототипа: `local-prototype-v0.4`.
- Рабочая ветка мультиплеерной подготовки: `multiplayer-foundation`.
- Клиент: `apps/client`.
- Сервер: `apps/server`.
- Общие типы: `packages/shared`.

## Управление в локальной игре

- `WASD` - движение.
- Мышь - направление корабля.
- ЛКМ - стрельба.
- `M` - большая карта.
- `F1` - debug-информация.
- `P` - админская панель в dev-режиме.
