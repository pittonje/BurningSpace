# BurningSpace

BurningSpace — первый локальный прототип браузерной космической аркады с видом сверху. Сейчас это однопользовательская клиентская основа: большая карта, управляемый корабль, плавная камера, стрельба, HUD, мини-карта и большая карта.

## Требования

- Node.js 18 или новее.
- npm.
- Современный браузер с поддержкой WebGL.

## Установка и запуск

```bash
npm install
npm run dev
```

Дополнительные команды:

```bash
npm run build
npm run preview
npm run typecheck
```

## Управление

- `W` — вверх.
- `S` — вниз.
- `A` — влево.
- `D` — вправо.
- Мышь — направление носа корабля.
- ЛКМ — стрельба.
- `M` — показать или скрыть большую карту.
- `F1` — показать или скрыть отладочную информацию.

## Что реализовано

- Мир размером `12000 x 12000` с процедурным звездным фоном.
- Декоративные туманности, координатная сетка, граница карты.
- Красная база в левом верхнем углу, синяя база в правом нижнем углу и нейтральная зона в центре.
- Корабль игрока с независимым движением по WASD, плавным поворотом к курсору и эффектом двигателя.
- Пул снарядов с cooldown, свечением и автоматическим исчезновением.
- Камера с плавным следованием и ограничением по границам мира.
- HUD, закрепленный относительно экрана.
- Мини-карта и увеличенная полупрозрачная карта по клавише `M`.
- Строгий TypeScript без UI-фреймворков.

## Пока не реализовано

- Многопользовательский сервер и синхронизация.
- Colyseus, WebSocket, аккаунты и база данных.
- NPC, урон, смерть, выбор точки возрождения.
- Аванпосты, прокачка и полноценная линия фронта.
- Баланс оружия и боевые цели.
