# Task Template

## Issue

Link or short description.

## Owner Agent

Primary responsible agent.

## Goal

Expected result.

## Context Documents

Relevant docs and code references.

## Allowed Paths

Files and directories the task may change.

## Forbidden Paths

Files and directories the task must not change.

## Acceptance Criteria

Observable completion criteria.

## Tests

Required automated and manual checks.

## Dependencies

Required previous tasks, decisions, or external access.

## Handoff Target

Who reviews or continues the work.
