# Responsibility Matrix

## Product Architect

- Owns design direction.
- Owns roadmap and acceptance criteria.
- Does not directly own production code changes.

## Primary Implementer

- Owns focused implementation.
- Adds or updates tests.
- Opens PR and writes handoff.

## Independent Reviewer

- Performs read-only review.
- Reports findings with severity and references.
- Does not silently edit the implementation.

## QA Agent

- Owns reproduction steps.
- Runs regression checks.
- Expands diagnostic scenarios.

## Visual Agent

- Owns concepts and asset specifications.
- Does not change gameplay code.

## Human Owner

- Approves final design.
- Owns credentials and production decisions.
- Approves merges and major roadmap changes.
